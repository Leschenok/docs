id: workmodel
title: Основная модель
---

**Основная модель** - итоговая модель интеграции, которая может быть активирована. Внесение изменений непосредственно в **основную модель** запрещено. Все необходимые корректировки **основной модели** выполняются через **пользовательскую модель** с последующей отправкой **пользовательской модели** в **основную**.

В **основной модели** можно выполнять только следующие действия:
1. Активация интеграции.
2. Деактивация интеграции.
3. Восстановление модели из хранилища.
